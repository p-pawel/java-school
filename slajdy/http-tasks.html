Zadanie 3

Zadanie 4

Zmodyfikuj serwer tak, aby obsługiwał również adres cars/:id, pod którym zwróci pojedynczy obiekt typu Car w formacie JSON, który będzie zawierać id zgodny z żądaniem.

Klasa Car może zawierać np. dwa pola: long id i String name.

Zadanie 7
